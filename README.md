# weatherLITE
A minimalist weather app built with Next.js 13 with TypeScript, Tailwind CSS for styling and [WeatherAPI](https://www.weatherapi.com/docs/) to fetch weather data.

</br>

## Tech Stack
1. Next.js 13
2. TypeScript support
3. Tailwind CSS for styling
4. Font Force library for external fonts
